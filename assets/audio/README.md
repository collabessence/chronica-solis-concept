# Audio - SFX

Dodaj pliki .wav lub .ogg i przypisz w edytorze do wezlow:

- **AttackSound** (HitboxComponent): atak
- **HitSound** (HurtboxComponent): trafienie
- **ParrySound** (Player): parry
- **DeathSound** (Player): smierc
- **CheckpointSound** (Checkpoint): aktywacja checkpointu
- **PickupSound** (TeethPickup): odbiór zebow

Sceny: Player.tscn, Enemy.tscn, Boss.tscn, Checkpoint.tscn, TeethPickup.tscn
